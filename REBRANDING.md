# Evolution of Naming: HCS/HDA to CCA/CCD

To better align with the core philosophy of this invention, the architecture has been officially rebranded.

### 🏷️ Current Naming (The Standard)
*   **CCA (Cellular Computing Architecture):** The overarching theory and framework specification.
*   **CCD (Cellular Computing Document):** The self-contained, parallel-execution file format (`.ccd` or `.hda.html`).

### 📜 Legacy Naming (Internal References)
You may still find the following terms within the codebase, binary signatures, or early specifications:
*   **HCS (Honeycomb Cell System):** Now referred to as **CCA**.
*   **HDA (Honeycomb Data Architecture):** Now referred to as **CCD**.

### 🔍 Why the change?
While the "Honeycomb" metaphor perfectly describes the hexagonal lattice and cellular structure, **Cellular Computing** more accurately describes the technical shift: a move away from linear, top-down execution towards distributed, spatial, and parallel computation where the document itself acts as the runtime environment.
